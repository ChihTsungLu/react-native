# Tech Stack
1. 使用 React Native 框架的跨平台移動應用程式，專注於簡化工作搜尋流程。
2. 串接 RapidAPI 以存取富有內容的工作資訊，包括職缺、公司詳細資訊和工資。
3. 實現了搜尋和篩選功能，提供使用者高度自訂的工作搜尋體驗。
4. 優化使用者介面以提供直觀且具有吸引力的用戶體驗。
5. 使用 Docker 技術，將應用程式及其相依性封裝到容器中，確保環境一致性。
